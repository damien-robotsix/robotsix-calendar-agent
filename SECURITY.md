# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

Only the latest release in the current minor series receives security
patches. When a new minor version is released, the previous minor is no
longer supported.

## Reporting a Vulnerability

To report a security vulnerability, please use the
[GitHub Security Advisories](https://github.com/damien-robotsix/robotsix-calendar-agent/security/advisories/new)
feature.

## Disclosure Policy

- **Acknowledgement within 5 business days** of a report being filed.
- **Fix target: 90 days** from acknowledgement. If the issue is complex,
  we will communicate an updated timeline.
- **Coordinated disclosure**: the fix will be released and an advisory
  published simultaneously. We ask that reporters refrain from public
  disclosure until the advisory is live.
- **Attribution**: reporters who follow responsible disclosure will be
  credited in the release notes and advisory (unless they request
  anonymity).
